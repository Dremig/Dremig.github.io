    Examples

    vectors

    #include<vector>
using namespace std;

int main(){
    vector<int> evens;
    evens.push_back(20);
    evens.push_back(22);
    evens.insert(evens.begin() + 4,5,10);

    for(int i = 0;i < evens.size(); i++){
        cout << evens[i] << " ";
    }
}

    认为是可伸缩的就行。

    push_back用于塞到尾部，insert指定插入位置，这里(...5, 10)表示插入了5个10。

    然而这样报错了，改成这样，使用迭代器：

    ...
for(vector<int>::iterator it = evens.begin();it != evens.end();it++){
    cout<< *it << " ";
}
...

    有点像在模仿指针，迭代器是需要解引用的。

    迭代器的作用就是让算法更加通用。

    还有一种方法，其实就是迭代器，编译器会将其翻译为迭代器的版本：

    for (int e: evens){
    cout << e << " ";
}

    底层还有一些接口：

    • element access:at, operator[], front, back
    • iterators: begin, end, cbegin, cend
    • capacity: empty, size, capacity, reserve
    • modifiers: clear, insert, erase(删除), push_back(最后插)

    对于vectors的iterato是可以用< end的，因为可以比较大小，但是list之类的不行。

    Using the map

    Look up by a key, return a value.

    e.g. map<string, string>

    #include<map>
#include<string>
using namespace std;

int main(){
    map<string, float> price;
    price["apple"] = 3;
    price["orangle"] = 5;
    price["banana"] = 1;

    string item;
    int total = 0;
    while(cin >> item)
        total += price[item];

    cout<< total << endl;
}

    输入什么tom jerry输出不太对，使用price[tom]这种操作会导致在map插入一个tom，值为0。

    使用c++20左右的标准可以使用price.contain(item)以实现判断。

    #include<map>
#include<string>
using namespace std;

int main(){
    map<string, int> word_map;
    for (const auto & w: {"we", "are", "not", "human", "we", "are", "robots", "!!!"})
        ++word_map[w];
}

    这可以实现词频的统计。

    然后直接用：

    for (const auto & pair : word_map)
    cout << "{" << pair.first << ":" << pair.second << "}";

    也可以使用叫什么结构绑定，使用类似结构：

    for (const auto & [word, count] : word_map)
    cout << "{" << word << ":" << count << "}";

    这样。

    Using the stack

    我们来试图实现一下它的一些接口：

    
class Stack{
public:
    virtual ~Stack() = default;
    virtual void push(int) = 0;
    virtual void pop() = 0;
    virtual int top() = 0;
    virtual bool empty() = 0;
};

template <typename T>
class c_stack : public Stack<T>{
public:
    T& top() override; { return c.back(); }
    bool empty () const override; { return c.empty(); }
    size_t size() const override; { return c.size(); }
    void push(const T& x) override; { c.push_back(x); }
    void pop() override; { c.pop_back(); }
private:
    vector<T> c;
}

    可以这样换用所有container:

    template <typename T, typename Container = std::deque<T>>
...
private:
    Container c;

    然后我们调用：

    c_stack<char, list<char>> st;
