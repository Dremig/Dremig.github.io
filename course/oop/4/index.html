    static local variable

    static关键字使得修饰的变量或函数的作用域仅限于定义它的文件。

    举个例子：

    #include<iostream>

void access_count(){
    static int count = 0;
    cout << count << endl;
}

// void access_count(){
//     int count = 0;
//     cout << count << endl;
// }

int main(){
    for(int i = 0; i < 5; i++){
        access_count();
    }
}

    加上static后，count的值会一直保留，因为放在全局数据区了，这样就可以持续维护，具有持久性。

    • 访问范围/作用域受限
    • 存储的永久性，其状态可以进行维持

    pointer

    string s = "hello world";
string *ps = &s;

    ps进行一次解引用之后就是s。

    以及一个地址的copy：

    以及reference:

    char c; // a character
char &rc = c; // a reference to c

    那么rc就是c的别名，rc和c指向同一个地址，本质就是同一个东西了。

    dynamic memory allocation

    • new：分配内存，并返回指向该内存的指针
    • delete
    #include<iostream>
#include<cstdlib>
using namespace std;

int main(){
    int *pa = new int(100);
    cout << *pa << endl;

    int *pb = new int[10];
    for (int i = 0; i < 10; i++){
        pb[i] = i;
    }
    for(int i = 0; i < 10; i++){
        cout << pb[i] << endl;
    }

    delete pa;
    delete [] pb;// 要是仅有delete pb将报error
}

    对于new，相较malloc多执行了构造函数，delete的时候，也执行了析构函数。

    #include<iostream>
using namespace std;


struct Student {
    int id;
    Student() {
        id = 0;
        cout << "Student::Student()" << endl;
        cout << id << endl;
    }
    ~Student() {
        cout << "Student::~Student()" << endl;
        cout << id << endl;
    }
};

int main(){

    Student *ps1 = (Student *)malloc(sizeof(Student));
    cout << ps1->id << endl;
    Student *ps2 = new Student();
    cout << ps2->id << endl;

    Student *ps3 = new Student[5];

    for(int i = 0; i < 5; i++){
        ps3[i].id = i;
    }

    delete ps1;
    delete ps2;
    delete [] ps3;
}

    这里可以看到结果如下：
