    Theoetical Computer Science

    upper bound lower bound:

    • 比如排序的至多O(nlogn)就够了，上界的研究属于算法
    • 推导性质需要明确分析的对象

    我们所谓的problem其实是一个function

    • function指的是数学中的概念，对应计算什么
    • program指的是function的实现

    字母表 Alphabet, a finite set of symbols, 可以是任何字符:

    • \(\sum = \{0, 1\}\)
    • \(\sum^2 = \{00, 01, 10, 11\}\)
    • ...
    • \(\sum^0 = \{e\}\) empty strings
    • \(\sum^* = \sum^0 \cup \sum^1 \cup \sum^2 \cup ...\) 闭包，所有字符串的集合
    • \(x = a_1a_2...a_n\), \(y = b_1b_2...b_n\), \(xy = a_1a_2....b_1b_2...\)
    • x is a prefix of y, if \(y = xz\), and \(z \in \sum^*\)

    Encoding

    • E: \(A \rightarrow \{0, 1\}^*\) 符合one to one(单射)

    • Natural numbers N

      • \[NtS(n) = \begin{equation} \begin{cases}0, when\ n = 0\\ 1, when\ n = 1, \\NtS(n//2)parity(n) \end{cases} \end{equation}\]
    • \(N × N\)：
      • \(E(1, 6) = 1110, E(3, 2) = 1110\)，这样会有单射问题
      • 解决直接拼接的"非单射"冲突，定义一个"逗号"，但是太长
      • 本质问题还是1的编码是3的prefix

    prefix-free encoding

    \(E(x)\) is not a prefix of \(E(x')\) for any \(x \neq x'\) in \(A\)

    • \(E: A \rightarrow \{0, 1\}\) prefix-free encoding
    • \(\overline{E}: A^* \rightarrow \{0, 1\}^*\)

    \(\overline{E}(a_1, a_2, ..., a_n) = E(a_1)E(a_2) ... E(a_n)\)

    定理：\(\overline{E}\)为单射，证明就是反证看prefix-free

    存在性：是否总能找到prefix-free encoding

    定理：如果\(E\)为单射\(A \rightarrow \{0, 1\}^*\), 那么存在prefix-free encoding\(\ \ \overline{E}\)

    • 方法：\(0 \rightarrow 00, 1 \rightarrow 11\)，在最后再添加一个01串：这样找到了长度不多于2倍加2的prefix-free encoding

    定理：如果存在one to one \(E: A \rightarrow \{0, 1\}^*\)，存在one to one \(E': A^* \rightarrow \{0, 1\}^*\)

    • 本质是对维数的升高，重要

    countable

    1. A is coutable
    2. A is finite or \(\exist bijection\ f: A \rightarrow N\) 双射
    3. \(\exist\) one to one \(E: A \rightarrow N\) 单射
    4. \(\exist\) onto \(h': N \rightarrow A\) 满射

    定理：\(\{0, 1\}^*\) is coutable

    定理：A is coutable 当且仅当 \(\exist\) one to one \(E: A \rightarrow \{0, 1\}^*\)

    • 证明：
      1. A is coutable
      2. 存在A到N的一个单射
      3. 存在N到\(\{0, 1\}^*\)的一个满射
      4. 传递性，存在A到\(\{0, 1\}^*\)的一个单射

    我们这门课的输入集合是可数的 / 可编码的，实数比较这类看起来自然的问题不在我们的考虑范围内。
